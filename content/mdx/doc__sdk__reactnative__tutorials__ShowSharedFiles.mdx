# Documentation

Sharing files

### Show Shared File For Peer

---

This guide explains how to show all the shared files for Peer (Conversation, Bubble) by using using `<SharedFiles />` component.

### Preamble

---

This tutorial will explain how to use the`<SharedFiles />` component that used to render the sharing files in Rainbow. Also, you can check [Shared Files Service](/docs/sdk/reactnative/api/SharedFilesService), to see all the supported shared file services.

---

### How To Use:

First you should fetch your shared files by calling the method `fetchAllSharedFilesInPeer(peerId: string, isBubble: boolean)` from `SharedFilesService`. And you need to listen for fetching the files result by using `EventType.GetAllSharedFileWithPeerResult` event. After you successfully got the shared files array you need to import the `<SharedFiles ..props />` from the RN sdk npm package and pass some requited props as explained above in the table. The `SharedFils` component will render the files list you share through the provided peer(conversation or bubble) using a *RecyclerListView*. if you want to customize the files list or you want to use another list view you can call the optional method `renderFilesItems()`.

You can check the file information details \[file size, name, extension, owner and the file viewers\] by clicking on file from the rendered list, then a new view will open shows all the file details. You can control on how you want to view these information by calling the method `onFileItemClicked?: (item: IFile)`

```tsx
import { IPeer, SharedFiles, IFile } from 'react-native-rainbow-module';

...
...
    const [sharedFileArray, setSharedFileArray] = useState<IFile[]>([]);
    const [isLoading, setIsLoading] = useState<boolean>(true);
    const isBubble = peer.type === PeerType.Bubble;

    useEffect(() => {
        sharedFilesService.fetchAllSharedFilesInPeer(peer.jId, isBubble);
        const sharedFileWithPeerResult = eventEmitter.addListener(
            EventType.GetAllSharedFileWithPeerResult,
            (eventData: IFile[]) => {
                console.log(`GetAllSharedFileWithPeerResult with file count ${eventData.length}`);
                setSharedFileArray(eventData);
                setIsLoading(false);

            }
        );

        return () => {
            sharedFileWithPeerResult.remove();
        }
    }, [isBubble, peer]);

const renderOnFileClicked = (selectedFile: IFile) => {
    // Go the the file description view, for example!
        Actions.FileDescription({ file: selectedFile });
    }
    const renderCustomFilesList = (files: IFile[]) => {
    // If you want another list than the RecyclerListView we used in the `SharedFiles` component to
    // render the shared files
        return <Text> you should pass your custom list here! </Text>;
    }

    return (
            <View style={{flex: 1}} >
                <SharedFiles
                peer={peer}
                onFileItemClicked={renderOnFileClicked}
                renderFilesItems={renderCustomFilesList}
                sharedFileArray={sharedFileArray}
                isLoading={isLoading}
                />
            </View>
    );
```
