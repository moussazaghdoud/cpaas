## getSuggestedContacts

This API allows to list users network suggestions(nbSuggestions, format) ⇒ `Promise.<Array.<Contact>>` Method getSuggestedContacts

**Kind**: global function  
**Returns**: `Promise.<Array.<Contact>>` - List of user's network suggestions  
**Access**: public

Param

Type

Description

nbSuggestions

`number`

\[optional\] Allow to specify the number of suggestions to retrieve. Default value: 50 Allowed values: \[1;50\]

format

`string`

\[optional\] Allows to retrieve more or less user details in response.  
\- small: id displayName companyName companyId jid\_im isTerminated lastName firstName  
\- medium: id displayName companyName companyId jid\_im jid\_tel isTerminated lastAvatarUpdateDate lastUpdateDate lastName firstName guestMode  
\- full: id displayName companyName companyId loginEmail nickName title jobTitle country language timezone jid\_im jid\_tel isTerminated lastAvatarUpdateDate lastUpdateDate roles phoneNumbers emails lastName firstName guestMode lastOfflineMailReceivedDate Default value: small Allowed values: small, medium, full

## getSuggestedContactsLargeCompany

This API allows to list users network suggestions(excludeIds, limit, offset, department, jobtitle, tags, more, format) ⇒ `Promise.<any>` Method getSuggestedContactsLargeCompany

**Kind**: global function  
**Returns**: `Promise.<any>` - List of user's network suggestions  
**Access**: public

Param

Type

Description

excludeIds

`Array.<string>`

\[optional\] Array of user ids to exclude from suggestions

limit

`number`

\[optional\] Allow to specify the number of suggestions to retrieve. Default value: 20

offset

`number`

\[optional\] Allow to specify the position of first suggestion to retrieve (first suggestion if not specified). Default value: 0

department

`boolean`

\[optional\] Allow to retrieve only department based suggestions. Default value: false

jobtitle

`boolean`

\[optional\] Allow to retrieve only jobtitle based suggestions. Default value: false

tags

`boolean`

\[optional\] Allow to retrieve only tags based suggestions. Default value: false

more

`boolean`

\[optional\] Allow to retrieve only default based suggestions (creation date). Default value: false

format

`string`

\[optional\] Allows to retrieve more or less user details in response.  
\- small: id displayName companyName companyId jid\_im isTerminated lastName firstName  
\- medium: id displayName companyName companyId jid\_im jid\_tel isTerminated lastAvatarUpdateDate lastUpdateDate lastName firstName guestMode  
\- full: id displayName companyName companyId loginEmail nickName title jobTitle country language timezone jid\_im jid\_tel isTerminated lastAvatarUpdateDate lastUpdateDate roles phoneNumbers emails lastName firstName guestMode lastOfflineMailReceivedDate Default value: small Allowed values: small, medium, full
