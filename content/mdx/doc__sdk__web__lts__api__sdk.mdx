## SDK

This is the glue that binds all the SDK for Web services together. From that entry class, you can access to all the services and events integrated into this SDK for Web. In addition to that, this module logs in the browser console some interesting information about the application environment.

**Access**: public

-   [SDK](#module_SDK)
    -   [version()](#module_SDK+version) ⇒ `string`
    -   [mode()](#module_SDK+mode) ⇒ `string`
