# Documentation

Conversation Service

### Conversation Services

---

### Preamble

---

The Conversations Service provides methods for managing conversations, including retrieving conversations, sending conversations by email, deleting conversation messages, and retrieving conversations by JID.

## Methods

### getConversations()

---

Retrieves all conversations.

-   Returns: `void`
-   Example:

```typescript
import { conversationsService } from 'react-native-rainbow-module'
...
const getConversationList = () => {
  conversationsService.getConversations();
}
// listen the conversation list result
eventEmitter.addListener(
      EventType.conversationsUpdated,
      (eventData: IConversation[]) => {
        // handle conversation list
      }
    );
```

-   Note: This method triggers the EventType.conversationsUpdated event.

### sendConversationByEmail(conversationJid: string)

Sends a conversation by email.

-   Parameters
    
    -   `conversationJid`: The JID of the conversation to send.
-   Returns: `void`
    
-   Example:
    

```ts
import { conversationsService } from 'react-native-rainbow-module'
...
const sendConversation = (conversationJid: string) => {
  conversationsService.sendConversationByEmail(conversationJid: string);
}
```

### deleteConversationMessages(conversationJid: string)

```
Deletes the messages of a conversation.
```

-   Parameters
    
    -   `conversationJid`: The JID of the conversation to delete messages from.
-   Returns: `void`
    
-   Example:
    

```ts
import { conversationsService } from 'react-native-rainbow-module'
...
const deleteConversation = (conversationJid: string) => {
  onversationsService.deleteConversationMessages(conversationJid: string);
}
```

### getConversationByJid(conversationJid: string)

Retrieves a conversation by JID.

-   Parameters
    -   `conversationJid`: The JID of the conversation to retrieve.
-   Returns: `void`
-   Example:

```ts
import { conversationsService } from 'react-native-rainbow-module'
...
const sendConversation = (conversationJid: string) => {
conversationsService.getConversationByJid(conversationJid);
```

### Events

The Conversations Service triggers the following events:

-   `EventType.conversationsUpdated`: Triggered when the conversation list is updated.
-   `EventType.conversationsUpdated`: Triggered when a conversation is deleted.
