# Enumeration: SearchBubblesScope

List of different ways to search for a bubble

## Enumeration Members

### SEARCH\_BY\_BUBBLE\_NAME

> **SEARCH\_BY\_BUBBLE\_NAME**: `"SEARCH_BY_BUBBLE_NAME"`

The bubble search must be performed on the name of the bubble

---

### SEARCH\_BY\_PARTICIPANT\_NAME

> **SEARCH\_BY\_PARTICIPANT\_NAME**: `"SEARCH_BY_PARTICIPANT_NAME"`

---

### SEARCH\_BY\_TAG\_NAME

> **SEARCH\_BY\_TAG\_NAME**: `"SEARCH_BY_TAG_NAME"`
