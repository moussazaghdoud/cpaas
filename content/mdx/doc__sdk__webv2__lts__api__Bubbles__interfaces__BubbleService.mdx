### searchBubbles()

> **searchBubbles**(`searchPattern`, `options`?): [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<[`BubbleSearchResult`](doc/sdk/webv2/lts/api/Bubbles/interfaces/BubbleSearchResult)\[\]>

This API allows to search Rainbow bubbles that match a search criteria applied on some name properties such as the name of the bubble or the names of bubble participants for example. One or more name properties can be optionally specified to define the scope of the search. If no scope is specified, the search is performed by default on the name of the bubble only. It is also possible to optionally specify how the list of bubbles returned as results should be sorted. By default, the bubbles list will be sorted by bubble name.

#### Parameters

• **searchPattern**: `string`

The search criteria

• **options?**: [`SearchBubblesOptions`](doc/sdk/webv2/lts/api/Bubbles/interfaces/SearchBubblesOptions)

Options to specify the scope of the search and how the results are to be returned.

#### Returns

[`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<[`BubbleSearchResult`](doc/sdk/webv2/lts/api/Bubbles/interfaces/BubbleSearchResult)\[\]>
