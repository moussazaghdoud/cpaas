# Managing Contacts

---

### Overview

The Contacts feature allows you to manage and interact with the user's contact list. This guide shows how to retrieve, display, and interact with contacts in your application.

## Features

-   Retrieve the complete contact list
-   Get real-time updates when contacts change
-   Access detailed contact information
-   Handle contact presence updates
-   Removing contacts they no longer wish to be connected with.

### How It Works

The contact management system is asynchronous. You will typically call a method from the `contactsService` to initiate an action (like fetching contacts), and then listen for a specific event to receive the result. This event-driven architecture ensures that your application's UI remains responsive while data is being fetched over the network.

## Basic Usage

### Retrieving Contacts

```tsx
import React, { useState, useEffect } from 'react';
import { View, Text, FlatList } from 'react-native';
import { eventEmitter, EventType, IContact, startUpService } from 'react-native-rainbow-module';


interface ContactItemProps {
  contact: IContact;
}

export const ContactsComponent: React.FC = () => {
  const [contacts, setContacts] = useState<IContact[]>([]);

  useEffect(() => {
    // Initial fetch of contacts
    startUpService.getRosterContacts();

    // Subscribe to contact updates
    const contactsUpdated = eventEmitter.addListener(
      EventType.ContactsUpdated,
      (updatedContacts: IContact[]) => {
        setContacts(updatedContacts);
      }
    );

    // Cleanup listener on component unmount
    return () => {
      contactsUpdated.remove();
    };
  }, []);

const renderItems = (item: IContact) => {
  return <ContactCardView contact={item} rightItem={contactCardRightComponent(item)} />;
}

  const renderEmptyList = () => (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text>No contacts found</Text>
    </View>
  );

  return(
    <Contacts
      contacts={contacts}
      renderItems={renderItems}
      renderEmptyList={renderEmptyList}
    />
  )
};
```
