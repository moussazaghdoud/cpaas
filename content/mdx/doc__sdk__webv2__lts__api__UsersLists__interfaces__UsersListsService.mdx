# Interface: UsersListsService

## Methods

### createUsersList()

> **createUsersList**(`profile`): [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`any`\>

Create a new list of users Only the name of the users list is mandatory. The other information (description and favorite status) are optional and can be set afterwards using the 'updateUsersListInfo' method. A ON\_USERS\_LIST\_CREATED event is sent afterwards at the completion of the operation to notify about the users list creation.

#### Parameters

• **profile**: [`UsersListProfile`](doc/sdk/webv2/lts/api/UsersLists/interfaces/UsersListProfile)

#### Returns

[`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`any`\>

### deleteUsersList()

> **deleteUsersList**(`listRef`): [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`any`\>

Delete a users list specified either by its reference or by its unique identifier. A ON\_USERS\_LIST\_DELETED event is sent afterwards at the completion of the operation to notify about the users list deletion.

#### Parameters

• **listRef**: `string` | [`UsersList`](doc/sdk/webv2/lts/api/UsersLists/interfaces/UsersList)
