# Documentation

Events

## Members

[Events](#Events)

This module handle every events that come from native Rainbow.  
To receive them, you need to subscribe individually to each of the following events see method provided by the class below.  

## Functions

[addListener(e, listener)](#addListener)

Adds a listener to be invoked when events of the specified type are emitted.The data arguments emitted will be passed to the listener function.

[removeListener(e, listener)](#removeListener)

Removes the given listener for event of specific type.

### addListener(e, listener)

---

Adds a listener to be invoked when events of the specified type are emitted.The data arguments emitted will be passed to the listener function.

**Kind**: global function **Access**: public

Param

Type

Description

e

`EventType`

Name of the event to listen to.

listener

`function`

Function to invoke when the specified event is emitted.

### removeListener(e, listener)

---

**Kind**: global function **Access**: public

Param

Type

Description

listener
