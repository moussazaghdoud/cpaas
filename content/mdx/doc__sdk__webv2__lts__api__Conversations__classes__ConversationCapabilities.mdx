# Class: ConversationCapabilities

This class is used to represent a conversation capabilities

## Properties

### addMedia

> **addMedia**: `boolean` = `false`

True if user is already in call and has an active fullJid

---

### mediaAvailable

> **mediaAvailable**: `boolean` = `false`

Means telephony || webRTC is TRUE

---

### sharedDesktop

> **sharedDesktop**: `boolean` = `false`

User can make a desktop sharing call with that user

---

### telephony

> **telephony**: `boolean` = `false`

User can make or receive telephony calls

---

### webRTC

> **webRTC**: `boolean` = `false`

User can make / receive webrtc calls
