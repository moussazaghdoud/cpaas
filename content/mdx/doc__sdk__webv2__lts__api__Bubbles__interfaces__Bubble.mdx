### delete()

> **delete**(`createArchive`?): [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<[`Bubble`](doc/sdk/webv2/lts/api/Bubbles/interfaces/Bubble)\>

If TRUE, the bubble will be archived for all users (i.e. read only mode for all users still inside);
