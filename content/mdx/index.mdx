## En savoir plus sur nos SDKs

![SDK pour CSharp](/static/media/sdk-csharp.941d117386cfa1da6cd40d58bd0b5c78.svg)

### SDK pour CSharp

Créez des applications de bureau qui connectent les personnes via un riche ensemble de services. Le SDK Rainbow pour CSharp vous fournira des discussions en direct avec les utilisateurs finaux, des conversations audio ou vidéo en "one-to-one" ou en groupe à l'aide de bulles.

[Vers le SDK CSharp](/docs/sdk/csharp)

![SDK pour Web](/static/media/sdk-web.42357f2729f73f555b60b998f0533af8.svg)

### SDK pour Web

Créez des applications Web pour votre site Web qui connectent les personnes via un riche ensemble de services. Le SDK Rainbow pour le Web fournira à votre site Web existant des discussions en direct avec les utilisateurs finaux, des conversations audio ou vidéo en "one-to-one" ou en groupe à l'aide de bulles.

[Vers le SDK Web](/docs/sdk/web)

![SDK pour Android](/static/media/sdk-android.331fb155bc224746dab454f7b8c0b7f2.svg)

### SDK pour Android

Etablissez des discussions audio et vidéo entre vos utilisateurs via votre application Android. Le SDK Rainbow pour Android fournit des services Rainbow à votre application mobile, comme discuter en direct en audio ou vidéo, discuter en groupe ou échanger des fichiers.

[Vers le SDK Android](/docs/sdk/android)

![SDK pour iOS](/static/media/sdk-ios.de2115d6a0c5496c8517bc5b9881cdb0.svg)

### SDK pour iOS

Etablissez des discussions audio et vidéo entre vos utilisateurs via votre application iOS. Le SDK Rainbow pour iOS fournit des services Rainbow à votre application mobile, comme discuter en direct en audio ou vidéo, discuter en groupe ou échanger des fichiers.

[Vers le SDK iOS](/docs/sdk/ios)

![SDK pour React Native](/static/media/sdk-reactNative.2406d58f50b14650e0138488d93d1483.svg)

### SDK pour React Native

Utilisez le SDK pour React Native pour développer des applications hybrides qui s'exécuteront à la fois sur iOS et Android. Il permettra d'intégrer rapidement des fonctionnalités de chat, de vidéo et de voix à vos applications mobiles.

[Vers le SDK ReactNative](/docs/sdk/reactnative/home)

![SDK pour Node.js](/static/media/sdk-nodejs.2d6249fb757bd8130e4ca92dc4eeba5a.svg)

### SDK pour Node.js

Créez un chatbot ou surveillez les alarmes et les statistiques de vos appareils IoT. Le SDK Rainbow pour Node.js est une bibliothèque JavaScript qui se connecte à Rainbow et fournit à votre application cloud ou IoT un agent toujours actif et capable de communiquer avec les utilisateurs ou les objets.

[Vers le SDK Node](/docs/sdk/node)

![API REST](/static/media/rest-api.e94b5ac454480bb4a8b67b612d797681.svg)

### API REST

Connectez vos applications pour les tâches d'exploitation administratives telles que l'approvisionnement et la gestion des compagnies et des utilisateurs Rainbow. En utilisant les API REST Rainbow, vous pouvez connecter vos applications front-end ou backend à Rainbow pour effectuer de telles tâches. L'outil Rainbow CLI (interface de commande en ligne) vous permet aussi d'effectuer ces actions très facilement!

[Vers l'API REST](/docs/apis/home)

    [S'authentifier !](/login)
