# Documentation

Limits

## Limits

---

The following table lists the limits that can apply to Rainbow services. Limits are grouped by theme.

These limits could be updated depending on the Rainbow platform evolutions.

### User and profile

---

Limit

Values

Details

**Max connections per user**

`5`

Same user account can be connected to Rainbow up to 5x (several resources) at the same time (ie: using several devices).  
Connecting more resources will disconnect existing ones.

**Max size of user's avatar**

`4 Mo`

The maximum size of the user's avatar that can be uploaded.

### Companies

---

Limit

Values

Details

**Max administrator accounts per company**

`60`

The maximum number of users having administrator rights (such as company\_admin) in a company.

**Max custom data size per company**

`5 Ko`

The maximum size of the custom data that can be attached to a company.

**Max size of company's avatar**

`4 Mo`

The maximum size of the company's avatar that can be uploaded.

**Max size of company's banner**

`10 Mo`

The maximum size of the company's banner that can be uploaded.

### Bubbles

---

Limit

Values

Details

**Max users per bubble**

`5000`

Number of users that can be part of a bubble.  
Once reached, remaining invitations will be refused for users joining the bubble.

**Max active bubbles created per user**

`1000`

Number of actives bubbles that can be created by a user at a time.  
A user can create more than 1000 bubbles only if he's owning less than 1000 actives bubbles.  
Archived owned bubbles are not taken into account.  

**Max active bubbles joined per user**

`200`

Number of bubbles that can be join by a user at a time.  
When joined, bubbles become actives and the user is notificed about incoming messages.

**Max custom data size per bubble**

`80 Ko`

The maximum size of the custom data that can be attached to a bubble.

**Max size of bubble's avatar**

`4 Mo`

The maximum size of the bubble' avatar that can be uploaded.

### Channels

---

Limit

Values

Details

**Max items saved per channel**

`100`

When connecting to Rainbow, only the last 100 items could be retrieved from a channel.  
Default value is 30.

**Max size of channel's notification**

`56 ko`

The maximum size of the channel's content which includes the title, the message, the url and any other extran content.

**Max size of channel's avatar**

`4 Mo`

The maximum size of the channel's avatar that can be uploaded.

### Files

---

Limit

Values

Details

**Max quota per user**

`1 Go`

Number of Go a user can store on Rainbow.  
Depends today on user profile only.

**Max size per file**

`100 Mo`

Maximum size of a file that can be shared in Rainbow

### Groups

---

Limit

Values

Details

**Max users per group**

`500`

Number of users that can be added to a group

**Max groups per user**

`50`

Number of groups a user can create

### Guests (CPaaS role)

---

Limit

Values

Details

**Max active guests per company**

`1000`

Number of active guests accounts that can exist at a time  
Existing expired guest accounts are not taken into account.

**Max time to live per guest**

`172800 s`

The maximum validity duration of a guest account (ie: 2 days). When expired, a guest account can't connect to Rainbow.  
If not set, time to live is set by default to 172800.

### Instant Messages

---

Limit

Values

Details

**Max message size**

`1 Ko`

Maximum size of a chat message (content) that can be sent through Rainbow.  
Longer messages have to be split.
