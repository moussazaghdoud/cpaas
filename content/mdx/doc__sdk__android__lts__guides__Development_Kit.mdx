# Documentation

Development Kit

LTS release 2.47.0Published on 2024-04-20

## Development Kit

---

The Rainbow SDK for Android contains:

-   An Android library: This library has to be included in your application via Gradle.
    
-   Several starter kit applications to demonstrate how to use the SDK for Android library.
    

### Android SDK module

---

The Rainbow Android SDK is available via Gradle. See the [Getting started](/docs/sdk/android) guide to understand how to load it.

### Starter kits

---

In order to simplify the SDK for Android development, some project or starter kit are available on Github.

You can have access to these samples by cloning the GitHub project: [Rainbow-Android-SDK-Samples](https://github.com/Rainbow-CPaaS/Rainbow-Android-SDK-Samples)

**NB: You have to create a Rainbow application and provide the identifier and the key in the `MyApp` class.**
