# Getting Started

Welcome to the Alcatel-Lucent Enterprise **Rainbow CSharp SDK WebRTC.Abstractions**

This SDK provides provides common interfaces and objects to implement/use to be compatible with WebRTC features provided by the [**Rainbow.CSharp.SDK.WebRTC** library](doc/sdk/csharpprevious/webrtc/lts/guides/001_getting_started?isBeta=true).

It cannot be used directly except if you want to create a new implementation to support WebRTC features in another environment.

## Nuget package

The SDK is provided as a Nuget package available [here](https://www.nuget.org/packages/Rainbow.CSharp.WebRTC.Abstractions/)

This package targets **.Net Standard 2.0**, **.Net Standard 2.1**, **.Net Core 6.0** and **.Net 8.0**

## Implementations available:

For the moment, there 2 differents implementations available:

-   [**Rainbow.CSharp.SDK.WebRTC.Desktop** library](doc/sdk/csharpprevious/webrtc.desktop/lts/guides/001_getting_started?isBeta=true) compatible with **Windows, MacOs, and Linux**
    
-   **Rainbow.CSharp.SDK.WebRTC.Unity** library (for the moment used in private usage only)
