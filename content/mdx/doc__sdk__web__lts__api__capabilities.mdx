LTS release 4.0.0Published on 2022-12-21

## Capabilities

This service manages the capabilities of the connected user. The capabilities allow to use the Rainbow features depending on the user subscription profile. The main methods proposed in that service allow to:  
\- Get all capabilities  
\- Check each capability individually

**Access**: public
