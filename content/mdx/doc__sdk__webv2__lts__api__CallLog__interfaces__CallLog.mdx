---

### state

> `readonly` **state**: `string`

The call state:

-   "answered": Refers to an answered call with a conversation established and this, regardless of direction (Incoming or outgoing).
-   "missed": Refers to an unanswered call, regardless of direction (Incoming or outgoing).
-   "forwarded": Refers to an incoming call which has been forwarded to the connected user (who was not the initial destination).

### userCallbackNumber?

> `optional` `readonly` **userCallbackNumber**: `string`

The number used to call back a Rainbow user if this one has not shared the private profile fields, including their telephone numbers. As a result, there is no telephone number visible (so usable) in the user's associated contact object. This number has been received as call information and is linked to the current call log only.
