# Documentation

Object IRainbowContact

LTS release 2.61.0Published on 202-04-06

## object IRainbowContact

### @Suppress (

Interface representing a Rainbow contact, extending the \[IDisplayable\] interface. It includes methods for retrieving various information and capabilities associated with a Rainbow contact.

### val id: String?

Gets the contact's ID.

The ID is non-null only for Rainbow contacts. For phone contacts or Office 365 contacts, this property returns null.

-   **Returns:** The contact's ID as a String or null.

### val jid: String?

Gets the contact's Jabber identifier (JID).

Jabber identifiers (JIDs) uniquely identify individual entities in the Jabber network. **See Also:** [Definition of Jabber Identifiers](https://xmpp.org/extensions/xep-0029.html)

-   **Returns:** The contact's JID as a String or null.

### val uniqueId: String?

Gets the contact's unique ID.

When the instance is a Rainbow contact, this method is equivalent to \[jid\]. Otherwise, for other Contact types, this method returns a unique ID.

-   **Returns:** The contact's unique ID as a String or null.

### val additionalInformation1: String?

Gets the first additional information associated with the contact.

This property is set for non-Rainbow users (office, company directory, personal contacts).

-   **Returns:** First additional information associated with the contact as a String or null.

### val additionalInformation2: String?

Gets the second additional information associated with the contact.

This property is set for non-Rainbow users (office, company directory, personal contacts).

-   **Returns:** Second additional information as a String or null.

### val isValidRainbowUser: Boolean

Indicates whether the contact is a valid Rainbow user.

This property checks if the contact is a valid Rainbow user and not from another directory (office, company directory, personal directory). If the contact is a Rainbow user, it further checks that they have a valid ID, a valid Jabber identifier (JID), and either a first name or a last name.

-   **Returns:** true if the contact is a valid Rainbow user with a valid ID, a valid JID, and either
    
    a first name or a last name; false otherwise.

-   **Returns:** The contact's company identifier as a String or null.

### val tags: List

Gets the contact's tags, if any.

This attribute represents an array of "free" tags associated with the user. A maximum of 5 tags is allowed, and each tag can have a maximum length of 64 characters. Tags are set by users who have administrator rights on the user. The user himself can't modify his tags. Tags are only visible to the user and all contacts belonging to his company. For other contacts, this method will return an empty list.

-   **Returns:** The contact's tags as a List or an empty List.
