# Enumeration: CallDirection

Call direction

## Enumeration Members

### INCOMING

> **INCOMING**: `"incoming"`

Incoming call.  
Call in which the connected user is the call recipient.

---

### OUTGOING

> **OUTGOING**: `"outgoing"`

Outgoing call.  
Call in which the connected user initiates the call.

---

> **UNDEFINED**: `""`

Default initial value.
