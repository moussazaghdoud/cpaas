## Favorites

This module is the basic module for handling Favorites in Rainbow. In Rainbow, Favorites are the way to list a most frequent, most used or the most important conversations, bubbles and bots. The main methods and events proposed in that service allow to:  
\- Create or delete a Rainbow Favorite (one-to-one, bubble or bot)  
\- Retrieve all information linked to that Favorite

**Emits**: `event:RAINBOW_ONFAVORITECREATED`, `event:RAINBOW_ONFAVORITEDELETED`
