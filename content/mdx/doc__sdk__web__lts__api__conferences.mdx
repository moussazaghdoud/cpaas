# Documentation

Conferences service

LTS release 4.0.0Published on 2022-12-21

## Conferences

The module is responsible for creating, joining and managing web conferences. The main methods and events proposed in that module allow to:  
\- Create and manage a web conference,  
\- Invite and manage participants  
\- Join a conference and manage different media streams

**Emits**: `event:RAINBOW_ONWEBCONFERENCEUPDATED`, `event:RAINBOW_ONWEBRTCMEDIAERROROCCURED`, `event:RAINBOW_ONBUBBLECONFERENCESTARTED`, `event:RAINBOW_ONBUBBLECONFERENCETALKERACTIVE`, `event:RAINBOW_ONBUBBLESHARINGCONFERENCESTARTED`, `event:RAINBOW_ONWEBCONFERENCEDELEGATED`  
**Access**: public
