This guide describes how to use SFU which is Rainbow conference service. This conference uses bubbles.

In this tutorial we will talking bout the following:

**How To Use:**

This example bellow will show to you how to use Conference Call in general and simple way. If you wanna more detail how to use it, you can check the [Sample Code](https://github.com/Rainbow-CPaaS/Rainbow-React-Native-Samples/tree/main/src/Calls/Conference/ConferenceCallComponent.tsx).

`Please Note:` in the `Sample Code` we listen beside the event `EventType.CurrentConferenceUpdated` we listen to `EventType.GetCurrentCall` event too. And the reason that we need to get the current call after retrieving the call view, please read this link for more info [Retrieve Active Call View](/docs/sdk/reactnative/tutorials/CurrentCallServices).

```tsx
import React, { FunctionComponent, useEffect, useState } from 'react';
import { eventEmitter, EventType, IConference, ConferenceActionsView, ConferenceCallView } from 'react-native-rainbow-module';
import { Body, Container, Header, Left, Title, Right, Icon } from 'native-base';
import {
    StyleSheet,
    View
} from 'react-native';

const ConferenceCallComponent: FunctionComponent = () => {

    const [conferenceCall, setConferenceCall] = useState<IConference>();

    useEffect(() => {
        // Listen to the conference updated including conference added, updated and removed
        const currentConferenceUpdatedListener = eventEmitter.addListener(EventType.CurrentConferenceUpdated, (call?: IConference) => {
            // incase of the removed call, the call event will be `undefined`
            setConferenceCall(call);
        });

    return () => {
            currentConferenceUpdatedListener.remove();
        }
    }, [conferenceCall]);

 // Check if there is a valid Conference Call to show it to the user.
    if (conferenceCall) {
        const { name } = conferenceCall.callPeer;

        return (
            <Container >
                <Header style={defaultStyle.headerColor} hasSegment={true}>
                    <Body>
                        <Title style={defaultStyle.headerTitle}>{name}</Title>
                    </Body>
                </Header>
                <View style={defaultStyle.backgroundImage}>
                    <ConferenceCallView conferenceCall={conferenceCall}  />
                    <ConferenceActionsView call={conferenceCall}  />
                </View>

            </Container>
        );
    }
    else {
        return null;
    }
}
```

**Conference Updated Event:**

To listen to the conference event changes, you can use `EventType.CurrentConferenceUpdated`, where the event response will be a conference object of type [IConference](/docs/sdk/reactnative/api/CallTypes) and the conference object will be as `undefined` in case of the `Call Removed`.

```tsx
const currentConferenceUpdatedListener = eventEmitter.addListener(EventType.CurrentConferenceUpdated, (call?: IConference) => {
            // incase of the removed call, the call event will be `undefined`
            setConferenceCall(call);
        });
```

**Conference Attendees Updates Event:**

To listen to the conference attendees event changes for example when one of the participant joining the conference or leave the conference, you can use `EventType.ConferenceAttendeesUpdates`, where the event response will be a attendees array of type [IConferenceParticipants](/docs/sdk/reactnative/api/CallTypes).

```tsx
const confAttendeesUpdatesListener = eventEmitter.addListener(EventType.ConferenceAttendeesUpdates, (attendees: IConferenceParticipants[]) => {
          // Check if there is a valid conference call object
            if (conferenceCall) {
                // update the conference call attendees with the new updated attendees.
                setConferenceCall({ ...conferenceCall, attendees });
            }
        });
```

**Conference Screen Sharing Updates Event:**

To listen to the conference Screen Sharing event changes for example when the screen sharing is enabled/disabled or when the participant who's sharing changed, you can use `EventType.GetConferenceScreenSharingUpdate`, where the event response will be as [ISharingConference](/docs/sdk/reactnative/api/CallTypes) type.

```tsx
const getSharingUpdate = eventEmitter.addListener(
            EventType.GetConferenceScreenSharingUpdate,
            (eventData: ISharingConference) => {
                console.log(`conference screen sharing update with isSharingEnabled: ${eventData.isSharingEnabled}`);
                // handle screen sharing updates
            }
        );
```

### Lock Conference Call

---

In case you wanna lock the conference call were to prevent any participant to join the conference call, you can use this method `conferenceService.lockConference(conferenceRoomId: string)` to switch the Lock state (lock/unlock), and then you can listen to the lock state changes result `EventType.GetConferenceLockState`.

Please check the following example:

```ts
import { conferenceService } from 'react-native-rainbow-module';

// Listen to the conference lock state event.
const conferenceLockState = eventEmitter.addListener(EventType.GetConferenceLockState, (eventData: boolean) => {
            console.log("conference lock state had changed to : ", eventData);
            ...
        });
        ...
// Switch conference lock state
const switchConferenceLock = (bubbleId: string) => {
    conferenceService.lockConference(bubbleId);
}
```

**Share Conference View:**

Incase you wanna add a Screen Sharing, you can use `ConferenceScreenSharingView` where you can depends on this Event: [EventType.GetConferenceScreenSharingUpdate](#EventType.GetConferenceScreenSharingUpdate) to show/hide the Screen Sharing view.

Please check the following props for your customization:

#

Name

Type

Description

1

style

ViewStyle

Your custom style for the screen sharing view

2

zOrder

number

the `zOrder` of the screen sharing view according to the other views

If you wanna more detail how to use it, you can check the [Sample Code](https://github.com/Rainbow-CPaaS/Rainbow-React-Native-Samples/tree/main/src/Calls/Conference/).

please check the following example:

```tsx
const [sharingInfo, setSharingInfo] = useState<ISharingConference>();
...

useEffect(() => {
...
// Listen to the Screen Sharing changes event
 const getSharingUpdate = eventEmitter.addListener(
            EventType.GetConferenceScreenSharingUpdate,
            (eventData: ISharingConference) => {
                setSharingInfo(eventData);
            }
        );
 ...

    }, []);

const shareConferenceView = ()=>{
    if(sharingInfo?.isSharingEnabled){
        return (
          <ConferenceScreenSharingView />
        );
    }
    else {
        return null;
    }
}
```
