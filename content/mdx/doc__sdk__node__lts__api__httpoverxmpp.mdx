### httPoverXMPP.get(urlToGet, headers, httpoverxmppserver\_jid) ⇒ `Promise.<any>`

This API allows to send a GET http request to an XMPP server supporting Xep0332.  

**Kind**: instance method of [`HTTPoverXMPP`](#module_HTTPoverXMPP)

**Returns**: `Promise.<any>` - An object of the result

**Category**: Rainbow HTTPoverXMPP

**Access**: public

**Nodered**: true

**Since**: 2.10.0

Param

Type

Default

Description

urlToGet

`string`

The url to request

headers

`Object`

`{}`

The Http Headers used to web request.

httpoverxmppserver\_jid

`string`

the jid of the http over xmpp server used to retrieve the HTTP web request. default value is the jid of the account running the SDK.
