#### Added

### \[2.3.0\] - 2025-09-01

#### Added

-   `IPeerConnection` interface: `RestartIce()` method

### \[2.2.0\] - 2025-07-07

#### Changed

-   Use `Microsoft.Extensions.Logging.Abstractions` **v9.0.0** (from v8.0.0)

#### Added

-   `IWebRTCFactory` interface: `SetLoggerFactory()`

---

### \[2.0.0\] - 2025-03-12

#### Added

#### Changed

### \[2.0.0-001-sts\] - 2024-07-01

#### Info

-   Contains all features of v1.X
-   New features will only been added in V2.X
