# Documentation

Invitation Service

# Invitation Services API Reference

---

This document provides a detailed API reference for managing invitations.

## Fetch All Invitations

Retrieves both sent and received invitations for the user.

**Action Dispatch:**

```typescript
import { getInvitations } from 'react-native-rainbow-module';

dispatch(getInvitations());
```

**Event Listeners:**

```typescript
import { eventEmitter, EventType, IInvitation } from 'react-native-rainbow-module';

// For received invitations
eventEmitter.addListener(
  EventType.ReceivedInvitationsUpdated,
  (invitations: IInvitation[]) => {
    console.log('Received invitations updated:', invitations);
  }
);

// For sent invitations
eventEmitter.addListener(
  EventType.SentInvitationsUpdated,
  (invitations: IInvitation[]) => {
    console.log('Sent invitations updated:', invitations);
  }
);
```

## Accept an Invitation

Accepts a received invitation.

**Action Dispatch:**

```typescript
import { acceptInvitation } from 'react-native-rainbow-module';

const invitationId = 'xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx';
dispatch(acceptInvitation(invitationId));
```

## Cancel an Invitation

## Resend an Invitation

Resends a sent invitation.

**Action Dispatch:**

```typescript
import { sendInvitationByEmail } from 'react-native-rainbow-module';

const userEmail = 'user@example.com';
dispatch(sendInvitationByEmail(userEmail));
```
