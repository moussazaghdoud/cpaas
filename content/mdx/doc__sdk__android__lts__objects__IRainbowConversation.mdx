# Documentation

Object IRainbowConversation

LTS release 2.61.0Published on 202-04-06

## object IRainbowConversation

### interface IRainbowConversation : IDisplayable

Created by letrongh on 27/03/2017.

### val id: String val peerId: String?

Get the id of this conversation.

-   **Returns:** the identifier of this conversation as a String

### val jid: String

Get the jid of the conversation depending on the type of the conversation (room or not).

-   **Returns:** the room jid if room type / the contact jid if not

### val isRoomType: Boolean

Allow to know if the conversation is a room type

-   **Returns:** true if the conversation is a room type

### val contact: IRainbowContact?

Get the contact of this conversation

-   **Returns:** a [IRainbowContact](/docs/sdk/android) object

### val lastMessage: IMMessage

Get the last message of this conversation.

-   **Returns:** the last message as a [IMMessage](/docs/sdk/android)

### val messages: ArrayItemList

Get the list of messages of this conversation

-   **Returns:** the list of messages as a [ArrayItemList](/docs/sdk/android)<[IMMessage](/docs/sdk/android)\>

### val room: Room?

Get the room of this conversation in case of room type

-   **Returns:** the room of this conversation in case of room type (null otherwise)

### val type: PeerType

Get the type of this conversation

-   **Returns:** USER, BOT or ROOM depends of the type of this conversation

### val unreadMessageNumber: Int

Get the number of unread messages of this conversation

-   **Returns:** the number of unread messages

### val isChatType: Boolean

Allow to know if the conversation is a chat type

-   **Returns:** true if the conversation is a chat type

### val mute: Boolean

Get the muted state of this conversation

-   **Returns:** true if the conversation is muted

### val isHistoryLoaded: Boolean

Allow to know if the conversation history is loaded

-   **Returns:** true if the conversation history is loaded

### val isWebinar: Boolean

Allow to know if the conversation belongs to a webinar

-   **Returns:** true if the conversation belongs to a webinar

### fun getMessageById (

Get a specific message by its id

-   **Returns:** a [IMMessage](/docs/sdk/android) the message with the specified id if any

### var textToSend: String

Get or set the last saved text message not already sent

-   **Returns:** the last saved message not already sent

### val bookmark: ConversationBookmark? fun registerChangeListener (ConversationListener) ConversationListener)

Get the bookmark of this conversation if there is one, null otherwise.

-   **Returns:** a [ConversationBookmark](/docs/sdk/android)

### val firstUnreadMessageId: String? }

Get the first unread message id of this conversation if there is one, null otherwise.

-   **Returns:** a [String](/docs/sdk/android)
