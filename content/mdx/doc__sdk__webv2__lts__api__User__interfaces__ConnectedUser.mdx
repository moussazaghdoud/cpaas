# Interface: ConnectedUser

This interface is used to represent a Rainbow user  
A User is defined by a set of public information (name, firstname, avatar...) and a set of private information that are only shared with contacts that are in the user's network or in the same company (email, phone numbers...)

Delete current user Account. This method deletes the connectedUser account and all its associated data. Depending on the configuration of the company hosting the connectedUser, this method may not work.

#### Returns

[`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`void`\>

---

### getAvailablePresenceStatusList()

> **getAvailablePresenceStatusList**(): `string`\[\]

Return a list of status that the user can currently Set (via changeMyPresence API); Values are \['online', 'away', 'xa', 'dnd'\]

-   online: user goes to online (or can 'exit') any manual state; In case of a call in progress, the user will go to busy state;
-   away: a manual Away state
-   xa: 'invisble' or 'appear offline' manual state
-   dnd: Do not disturb manual state

#### Returns

`string`\[\]

Returns an array of all available values to be used if the user wants to change his presence
