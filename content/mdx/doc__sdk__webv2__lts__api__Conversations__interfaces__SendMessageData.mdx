# Interface: SendMessageData

## Properties

### additionalContent?

> `optional` **additionalContent**: `MessageAdditionalContent`

Additional content (like markdown or vcard) to be added to the message

---

### mentions?

> `optional` **mentions**: `Mention`\[\]

The list of mentions inside the message

---

### repliedMessage?

> `optional` **repliedMessage**: `Message`

In case we want to reply to a given message

---

### urgency?

> `optional` **urgency**: `Urgency`

The urgency of the message; Allowed values are (standard) 'std', (notify) 'low', (warning) 'middle', (emergency) 'high'; Default value is "std"

---

### urgencyAckType?

> `optional` **urgencyAckType**: `"ack"` | `"ignore"`

In case of urgency ACK message, the type of the ACK to send; Accepted values are "ack" or "ignore"

---

### urlMetadata?

> `optional` **urlMetadata**: `UrlMetaData`

URL Meta data in case of sending URL
