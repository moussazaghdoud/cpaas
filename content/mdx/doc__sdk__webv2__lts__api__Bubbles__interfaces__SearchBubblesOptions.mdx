# Interface: SearchBubblesOptions

## Properties

### scope?

> `optional` **scope**: [`SearchBubblesScope`](doc/sdk/webv2/lts/api/Bubbles/enumerations/SearchBubblesScope)\[\]

Options to specify the scope of the search, i.e. the 'name' properties to be used as scope for the search The name properties that can be specified are:

-   The name of the bubble
-   The name of tag(s) set for the bubble
-   The name of bubble participants. One or more name properties can be optionally specified to define the scope of the search. If no scope is specified, the search is performed by default on the name of the bubble only.
