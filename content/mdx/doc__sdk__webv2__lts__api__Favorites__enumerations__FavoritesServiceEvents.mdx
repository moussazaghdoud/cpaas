# Enumeration: FavoritesServiceEvents

Favorites service events send to other services Can listen to it via `FavoritesService.subscribe()` API

## Events

### ON\_FAVORITES\_ORDER\_CHANGED

> **ON\_FAVORITES\_ORDER\_CHANGED**: `"ON_FAVORITES_ORDER_CHANGED"`

This `RBEvent` is sent when a favorite has been moved in the list.

#### Event Data

favorite - The favorite that has been moved in the list (as Favorite)

#### Event Data

position - The new position (starting from 0) of this favorite in the list

---

### ON\_FAVORITE\_CREATED

> **ON\_FAVORITE\_CREATED**: `"ON_FAVORITE_CREATED"`

This `RBEvent` is sent when a new favorite has been created.

#### Event Data

favorite - The new created favorite (as Favorite)

---

### ON\_FAVORITE\_DELETED

> **ON\_FAVORITE\_DELETED**: `"ON_FAVORITE_DELETED"`

This `RBEvent` is sent when a favorite has been deleted.

#### Event Data

id - The id (as string) of the favorite that has been deleted
