# Interface: DirectorySearchResults

List of data provided as the result of a directory search.

## Properties

### moreResultsAvailable

> **moreResultsAvailable**: `boolean`

Indicates whether there are more results available. If this flag is true and the contact/user searched for is not present in the results list, then the user can

-   request more results (via 'limit' option) for the same query
-   refine the search

---

### users

> **users**: [`User`](doc/sdk/webv2/lts/api/User/interfaces/User)\[\]

The list of users (as User model) matching the search criteria.
