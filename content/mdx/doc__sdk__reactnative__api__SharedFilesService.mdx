Shared Files Service

### Shared Files Services

---

### Preamble

---

The message services module `sharedFilesService` has all the methods supported by this library.

---

The following are all the supported services for the Shared Files:

fetchFileSentInBubble(bubbleId: string): void

### Fetch Files In Peer:

To fetch all the shared files in peer (Conversation, Bubble) you can use `sharedFilesService.fetchAllSharedFilesInPeer(peerJid: string, isBubble: boolean)`, where you can get the files results as array of[IFile](/docs/sdk/reactnative/api/IFile) type by listening to `EventType.GetAllSharedFileWithPeerResult`.

```ts
import { sharedFilesService, IFile } from 'react-native-rainbow-module'
...
...
sharedFilesService.fetchAllSharedFilesInPeer(peer.jId, isBubble);
        const sharedFileWithPeerResult = eventEmitter.addListener(
            EventType.GetAllSharedFileWithPeerResult,
            (eventData: IFile[]) => {
                logger.info(`GetAllSharedFileWithPeerResult with file count ${eventData.length}`);
                // Handle the files result
            }
        );
```

### Load More Files:

To Fetch more shared files in this peer (Conversation, Bubble), you can call `sharedFilesService.loadMoreSharedFiles(peerJid: string, isBubble: boolean, numberOfFiles: number = 10)`, where the default umber of files to fetch is 10 and you can get the files results as array of[IFile](/docs/sdk/reactnative/api/IFile) type by listening to `EventType.GetAllSharedFileWithPeerResult`.

```ts
import { sharedFilesService, IFile } from 'react-native-rainbow-module'
...
...
sharedFilesService.loadMoreSharedFiles(peer.jId, isBubble);
        const sharedFileWithPeerResult = eventEmitter.addListener(
            EventType.GetAllSharedFileWithPeerResult,
            (eventData: IFile[]) => {
                logger.info(`GetAllSharedFileWithPeerResult with file count ${eventData.length}`);
                // Handle the files result
            }
        );
```

### Fetch Sent Files In Bubble:

To Fetch `sent` the files in the bubble, you can use `sharedFilesService.fetchFileSentInBubble`, where you can get the files results as array of[IFile](/docs/sdk/reactnative/api/IFile) type by listening to `EventType.GetSharedFileSentWithBubbleResult`.

```ts
import { sharedFilesService, IFile } from 'react-native-rainbow-module'
...
const fetchSentFilesInBubble = (bubbleId: string)  => {
sharedFilesService.fetchFileSentInBubble(bubbleId);
}
// Listen to fetching file result
eventEmitter.addListener(
  EventType.GetSharedFileSentWithBubbleResult,
  (eventData: IFile[] ) => {
    //handle the shared files
  }
);
```

### Download File:

To download a file from a conversation, you can call `downloadFile(fileDescriptorId: string)`, where you can get the file download results by listening to `EventType.FileDownloadFinished`. You can also get the progress percentage of the download using `EventType.fileDownloadProgressUpdated` event.

```ts
import { sharedFilesService } from 'react-native-rainbow-module'
...
const downloadFile = (fileDescriptorId: string)  => {
sharedFilesService.downloadFile(fileDescriptorId);
}
// Listen to file download finish result
eventEmitter.addListener(
  EventType.FileDownloadFinished,
  (eventData: { isSuccess: boolean, fileId: string, errorMsg?: string }) => {
    //handle downloaded file
  }
);
```

### Delete File:

To delete a file from a conversation, you can call `deleteFile(fileDescriptorId: string)`.

```ts
import { sharedFilesService } from 'react-native-rainbow-module'
...
const deleteFile = (fileDescriptorId: string)  => {
sharedFilesService.deleteFile(fileDescriptorId);
}
```
