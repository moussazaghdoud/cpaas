## IPeer : `Interface`

IPeer interface is the parent type for each of [IContact](/docs/sdk/reactnative/api/IContact), [IBubble](/docs/sdk/reactnative/api/IBubble#IBubbles) and [IConversation](/docs/sdk/reactnative/api/IConversation) , which means its have all the common properties between those types, furthermore, you can pass any of the above types as IPeer because its the general type of those types.
