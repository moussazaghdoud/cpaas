### RAINBOW\_ON\_INVITATIONS\_LIST\_UPDATED

> **RAINBOW\_ON\_INVITATIONS\_LIST\_UPDATED**: `"RAINBOW_ON_INVITATIONS_LIST_UPDATED"`

### RAINBOW\_ON\_INVITATION\_ACCEPTED

> **RAINBOW\_ON\_INVITATION\_ACCEPTED**: `"RAINBOW_ON_INVITATION_ACCEPTED"`

This RBEvent is send when the invitation (from / to) contact is accepted with data {invitation:Invitation, user:User}

### RAINBOW\_ON\_INVITATION\_RECEIVED

> **RAINBOW\_ON\_INVITATION\_RECEIVED**: `"RAINBOW_ON_INVITATION_RECEIVED"`

This RBEvent is send when a new network invitation is received from another Rainbow user with data {invitation:Invitation, user:User}
