# Documentation

Manage conversations

### Retrieve Conversation

---

This guide explains how to retrieve the list of conversations (P2P or Bubble).

### Preamble

---

This guide will show you how to retrieve the list of conversations in your network.

To retrieve all active conversations (one-to-one and bubbles), all you need to do is import the component `<Conversations />` from the Rainbow RN library. Then you need to add some props as shown in the following table.

**How to use:**

```tsx
import React, { FunctionComponent, useEffect, useState } from 'react';
import { Content, Text } from 'native-base';
import { IBubble, conversationsService, IConversation, eventEmitter, EventType, Conversations, ConversationCard } from 'react-native-rainbow-module';
import { Actions } from 'react-native-router-flux';
import { Strings } from '../resources/localization/Strings';

export const ConversationsComponent: FunctionComponent = () => {
    const [conversations, setConversations] = useState<IConversation[]>([]);
    useEffect(() => {
    //Get all conversations using conversations services
    conversationsService.getConversations();
    //Listen to conversation list changes
    const conversationsUpdated = eventEmitter.addListener(
      EventType.ConversationsUpdated,
      (eventData: IConversation[]) => {
        setConversations(eventData);
      }
    );
    return () => {
      conversationsUpdated.remove();
    };
  }, []);

  const onClickItem = (conversation: IConversation) => {
    if (conversation.bubble) {
      const bubble: IBubble = conversation.bubble
      Actions.BubbleChatView({
        bubble
      });
    }
    else {
      Actions.PeerConversationChatView({
        conversation,
      });
    }
  };

  const renderEmptyList = () => {
    return (
      <Content>
        <Text note={true} style={defaultStyle.NoDataMessages}>
          {Strings.noDataFound}
        </Text>
      </Content>
    );
  }

  const renderItems = (item: IConversation) => {
    return <ConversationCard conversation={item} onClickItem={onClickItem}  />
  }

  return (
    <Conversations
      conversations={conversations}
      renderItems={renderItems}
      renderEmptyList={renderEmptyList}
    />
  );
}
```

**ConversationCard:**

We make a Conversation Card view just to make it easy to you to build the conversation list by using the `renderItems` method inside the Conversations list, please check the following props for the Conversation Card:
