# Documentation

Module RBLog

LTS release 2.61.0Published on 202-04-06

## module RBLog

### object RBLog

RBLog is used to print logs into logcat and also to write it to text files. You can get a zipped file of the logs by calling `zipLogsToPath()` method.

### @JvmOverloads fun initLogger (Context, DEBUG)

Initialize the logger with a specific logs level. This method is automatically called into `RainbowSdk.instance().initialize()`, so in most cases, you don't have to call it explicitly. Only call explicitly this method if you want to log with write to file before you have called `RainbowSdk.instance().initialize()`.

Param

Type

Description

context

Context

The context

level

Level

Can be one of `Level` enum: `TRACE`, `DEBUG`, `INFO`, `ERROR`, `WARNING`
