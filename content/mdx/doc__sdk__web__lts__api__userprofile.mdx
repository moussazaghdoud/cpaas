User profile service

LTS release 4.0.0Published on 2022-12-21

## UserProfile

This service manages with your Rainbow user profile and the presence of your contacts. The main methods and events proposed in that service allow to:  
\- Updating the user firstname and lastname,  
\- Updating the user phone number,  
\- Updating the user avatar (based on a file to send)
