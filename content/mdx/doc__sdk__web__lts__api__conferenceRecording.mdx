Conference Recording service

LTS release 4.0.0Publiée le 2022-12-21

## ConferencesRecordingService

The module is responsible for recording web conferences. The main methods and events proposed in that module allow to:  
\- Start recording  
\- Pause / Resume recording  
\- Stop recording  
\- Get My recording files  
\- Get recording files that are shared with me (in bubbles/rooms)
