# Interface: User

This interface is used to represent a Rainbow user  
A User is defined by a set of public information (name, firstname, avatar...) and a set of private information that are only shared with contacts that are in the user's network or in the same company (email, phone numbers...)

### subscribe()

> **subscribe**(`handler`, `filterEvent`?): `Subscription`

Subscribe to updates on the current contact (all events are of RBEvent type {name: string, data:any});

#### Parameters

• **handler**: `any`

The call-back function that will be subscribed to the RxJS subject

• **filterEvent?**: `string`\[\]

A list of event names that the user wants to receive when subscribing (used a filter)
