### globalSearch()

> **globalSearch**(`searchedText`, `limit`): [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`SearchTextConvResults`\>

Method used to search for text in all rooms and conversation. The searched text length should be 3+ characters. This API will return a list of all conversations / rooms with the number of occurances of the searched text in each one. If you need to get the results for a given conversation/room only, then use this API: searchForTextInConversation

#### Parameters

• **searchedText**: `string`

the text to be searched in all conversations and rooms

• **limit**: `number`

the maximum number results to take. Default value is 500;

#### Returns

[`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`SearchTextConvResults`\>
