## HuntingGroup

**Kind**: global class

**Access**: public

**Properties**

Name

Type

Description

name

`String`

Hunting group name - displayed on the caller phone set

shortNumber

`String`

Internal Number of the new cloud PBX hunting group

type

`String`

(hunting\_group) Group type. hunting\_group,manager\_assistant  
  
Note that **manager\_assistant** is **DEPRECATED**. See manager\_assistant subType  

subType

[`HuntingGroupSubType`](#HuntingGroupSubType)

(default) Group sub type, only Voice attendant users can be part of hg\_attendant group and can not be part of default group

policy

[`HuntingGroupPolicy`](#HuntingGroupPolicy)

(parallel) Hunting group policy - only parallel policy for hg\_attendant hunting group, only serial for manager\_assistant

timeout

`5-30`

(10) Timeout in seconds after which the next member of the hunting group will be selected (ringing) - applicable to serial or circular hunting group type only

isEmergencyGroup

`Boolean`

(false) Is this Hunting group the emergency group of the associated Cloud PBX

isEmptyAllowed

`Boolean`

(true) Indicates if the last active member of the hunting group can leave the group or not

isDDIUpdateByManagerAllowed

`Boolean`

(true) Indicates if changing the DDI of this hunting group by a manager is allowed or not

members

[`Array.<GroupMember>`](#GroupMember)

List of group members. In case of serial hunt group policy, or for several assistants, the order is the ringing order
