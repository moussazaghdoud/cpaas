# Documentation

CallsLog service

LTS release 4.0.0Published on 2022-12-21

## CallsLog

This service allow to get the call log and manage it. The main methods and events proposed in that service allow to:  
\- Get all calls log  
\- Delete one or all calls log  
\- Mark calls as read / unread

**Emits**: `event:RAINBOW_ONCALLLOGUPDATED`, `event:RAINBOW_ONCALLLOGACKUPDATED`  
**Access**: public

### callsLog.fetchCallLogs(numberOfCallLogs) ⇒ `Promise`

Get the Call-logs for current user; If this is the first time, the API will load the latest 'numberOfCallLogs' entries for this user Otherwise it will give all the available already loaded call-logs for this user; The user is able to ask for more call-logs via fetchPaginatedCallLogs API;

**Kind**: instance method of [`CallsLog`](#module_CallsLog)  
**Returns**: `Promise` - Promise with an array of call log entry  
**Access**: public

Param

Type

Description

numberOfCallLogs

`number`

Specify number of call logs to fetch from the server. Defaults to 100. (max value allowed 250)

### callsLog.fetchPaginatedCallLogs(numberOfElementsPerPage) ⇒ `Promise`

Get all available call-logs already loaded + the new fetched from the server

**Kind**: instance method of [`CallsLog`](#module_CallsLog)  
**Returns**: `Promise` - Promise with an array of call log entry  
**Access**: public

Param

Type

Description

numberOfElementsPerPage

`number`

Specify number of call logs to fetch from the server. Defaults to 100. (max value allowed 250)

### callsLog.getMissedCallLogCounter() ⇒ `number`

Get the number of call missed (state === "missed" && direction === "incoming")

**Kind**: instance method of [`CallsLog`](#module_CallsLog)  
**Returns**: `number` - The number of call missed  
**Access**: public  

### callsLog.getMissedCallLogNumber() ⇒ `number`

Get the number of call missed (state === "missed" && direction === "incoming") You can call this method before retrieving all the callLogs from the server (by using fetch)

**Kind**: instance method of [`CallsLog`](#module_CallsLog)  
**Returns**: `number` - The number of call missed  
**Access**: public  

### callsLog.deleteOneCallLog(id) ⇒ `boolean`

**Kind**: instance method of [`CallsLog`](#module_CallsLog)  
**Returns**: `boolean` - true  
**Access**: public

### callsLog.isInitialized() ⇒ `boolean`

**Kind**: instance method of [`CallsLog`](#module_CallsLog)  
**Returns**: `boolean` - True if the call logs have been retrieved. False elsewhere.
