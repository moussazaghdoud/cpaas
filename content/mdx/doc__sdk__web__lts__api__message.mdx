## MessageSDK

### MessageSDK~type : `string`

Message type. Can be one of the following values: - "Chat" - "File" - "FileSharing" - "FileSharing" - "WebRTC CAll" - "Recording" - "FORM" - "FORWARD"

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true

The message content

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true  

### MessageSDK~date : `Date`

Message date

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true  

### MessageSDK~senderId : `string`

ID of the Rainbow user who sent the message

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true  

### MessageSDK~isForwarded : `boolean`

Boolean indicating if the message was forwarded

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true  

### MessageSDK~receiptStatus : `number`

Status of the receipt Can be one of the following values:

-   0 - No receipt received yet
-   1 - No receipt received after a while, the server doesn't answer
-   2 - Receipt in progress
-   3 - The server has confirmed the reception of the message
-   4 - The message has been received but not read
-   5 - The message has been read

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true  

### MessageSDK~status : `string`

Status of the message

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true  

### MessageSDK~urgency : `string`

Urgency level. Can be one of the following values: 'std', 'low', 'middle', 'high'

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true  

### MessageSDK~shortFileDescriptor : `object`

Short file descriptor - if file is available and attached to the message. Used to download files.

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true  

### MessageSDK~fileId : `string`

ID of the file (if any)

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true  

### MessageSDK~messageHistory : `Array`

Array representing the history of message modifications

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true  

### MessageSDK~side : `string`

Indicator showing where is the message coming from. Possible values: "L" - message comes from other user, "R" - message comes from connected user, "ADMIN" - message comes from admin

**Kind**: inner property of [`MessageSDK`](#module_MessageSDK)  
**Access**: public  
**Read only**: true
