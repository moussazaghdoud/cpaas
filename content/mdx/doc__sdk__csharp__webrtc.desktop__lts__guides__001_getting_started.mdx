# Getting Started

Welcome to the Alcatel-Lucent Enterprise **Rainbow CSharp SDK WebRTC.Desktop**

This SDK provides additional services / tools for these platforms: **Windows, MacOs, and Linux**.

**Only x64 platform is supported.**

It uses as dependencies **Rainbow.CSharp.SDK**, **Rainbow.CSharp.SDK.Medias** and **Rainbow.CSharp.SDK.WebRTC.Abstractions** libraries.

Te be more precise this library is a specific implementation of **Rainbow.CSharp.SDK.WebRTC.Abstractions** which target **Desktop** platforms (as opposed to mobile platforms like Android or iOS).

## Nuget package

The SDK is provided as a Nuget package available [here](https://www.nuget.org/packages/Rainbow.CSharp.SDK.WebRTC.Desktop/)

This package targets **.Net Standard 2.0**, **.Net Standard 2.1** and **.Net 8.0**

## WebRTC features

Since this library is a implementation of **Rainbow.CSharp.SDK.WebRTC.Abstractions**, it provides same features than the ones described in [**Rainbow.CSharp.SDK.WebRTC** library](doc/sdk/csharp/webrtc/lts/guides/001_getting_started?isBeta=true). So please refer to the documentation of this library

## Third-Party Libraries

This implementation is using [**Rainbow.CSharp.SDK.Medias** library](doc/sdk/csharp/medias/lts/guides/001_getting_started?isBeta=true). So please refer also to the documentation of this library to know how to use it to manage medias and alos to know which third party libraires must be installed.
