LTS release 4.0.0Published on 2022-12-21

## Groups

This service manages user groups in Rainbow. Each user can create groups and invite other Rainbow users to be part of it. Without being invited, users can join a bubble. The main methods proposed in that module allow to: - Get all groups of the user - Create a new group - Add a contact in a group - Remove a contact from a group - Delete an existing group

**Emits**: `event:RAINBOW_ONGROUPCREATED`, `event:RAINBOW_ONGROUPUPDATED`, `event:RAINBOW_ONGROUPDELETED`, `event:RAINBOW_ONGROUPUSERADDED`, `event:RAINBOW_ONGROUPUSERREMOVED`  
**Access**: public
