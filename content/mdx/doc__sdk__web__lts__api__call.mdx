### Call()

This class represents a call.  
A call is a "short" interaction between the user and one or several contacts (Rainbow users or not) that uses either a WebRTC media (audio, video or screen sharing) or a Telephony media (PBX phone).  
No Rainbow users such as external users can only be engaged into a Telephony call.  
A call is always attached to a conversation.

## .isHuntingGroupCallSipWise() ⇒ `boolean`

For a sipwise (cloud tel) incoming call to an user who is belong to a hunting group We have to known if the call is for the group or privatly for the user This methode gives this info.

**Kind**: instance function  
**Returns**: `boolean` - Return true if the call is for the hunting group false otherwise  
**Access**: public  

## .getHuntingGroupCalledInfo() ⇒ `object`

For a sipwise (cloud tel) incoming call, if the call is for the group who the user is belong (meaning isHuntingGroupCallSipWise() return true) then this methode return the data to be used for called info.

**Kind**: instance function  
**Returns**: `object` - Return called with called: { phoneNumber: string, displayName: string} or null if isHuntingGroupCallSipWise() = false  
**Access**: public
