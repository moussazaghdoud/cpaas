### \[0.0.1\] - STS - 2023-05-26

---

Define interfaces and objects as an common abstraction to offer WebRTC features in different implementation.

For the moment, two implementations are available.

-   **Desktop WebRTC implementation**
    
    -   Based on **SipSorcery**, **FFmpeg** and **SDL2**
    -   Targets **Windows, MacOS** and **Linux**
-   **Unity WebRTC implementation** to use Rainbow features (with WebRTC) in **Unity project**
    
    -   **Unity** use the WebRTC implementation provided by the Chromium project.
    -   Targets Android, iOS but also Windows, Mac, Linux, Extended Reality devices (XR), Consoles (PS5, XBox, Switch), WebGL
