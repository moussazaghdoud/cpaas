## Methods and Events available

These tables permit to know also features available in XMPP event mode (default) or in S2S event mode. (Cf. [Server to server and XMPP events mode](/docs/sdk/csharp))

You can also download the [Excel file](doc/sdk/csharpprevious/core/lts/guides/file/SDK%20CSharp%20-%20XMPP%20vs%20S2S%20mode.xlsx) to have same info offline.
