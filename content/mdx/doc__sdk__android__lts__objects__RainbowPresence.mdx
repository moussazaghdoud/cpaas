# Documentation

Object RainbowPresence

LTS release 2.61.0Published on 202-04-06

## object RainbowPresence

### enum RainbowPresence

This class describes the RainbowPresence object.

### boolean isOffline()

Allow to know if the presence of the user is on **offline**

-   **Returns:** true if the user is offline

### boolean isOnline()

Allow to know if the presence of the user is on **online**

-   **Returns:** true if the user is online

### boolean isAway()

Allow to know if the presence of the user is on **away**.

This state is automatically set after a few times of inactivity.

-   **Returns:** true if the user is away

### boolean isManualAway()

Allow to know if the presence of the user is on **manual\_away**.

That means the user sets himself to this presence.

-   **Returns:** true if the user has manually set that he is away

### boolean isDND()

Allow to know if the presence of the user is on **dnd**.

-   **Returns:**

### boolean isDNDPresentation()

Allow to know if the presence of the user is on **dnd\_presentation**.

That means the user is on full screen mode on a computer.

-   **Returns:** true if the user is in presentation

### boolean isDNDCalendar()

Allow to know if the presence of the user is on **dnd\_calendar**.

That means the user is out of office or in a meeting and has chosen to have his presence set to DND automatically when that happens.

-   **Returns:** true if the user is in presentation

### boolean isMobileOnline()

Allow to know if the presence of the user is on **mobile\_online**.

That means the user is not online (but could be away) on desktop client, and connected on mobile.

-   **Returns:** true if the user is only online on mobile

### boolean isBusy()

Allow to know if the presence of the user is on **busy**.

-   **Returns:**

### boolean isUnsubscribed()

Allow to know if the presence of the user is on **unsubscribed**.

That means the connected user has not the other user is his network.

-   **Returns:**

### String getPresence()

Get the presence

-   **Returns:** the presence as a String
