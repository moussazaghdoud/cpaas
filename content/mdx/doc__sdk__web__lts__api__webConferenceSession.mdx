**Access**: public

### webConferenceSession.getSharingParticipant() ⇒ `WebConferenceParticipant`

Get web conference participant object that is currently sharing in the conference (could also be myself)

**Kind**: instance method of [`WebConferenceSession`](#module_WebConferenceSession)  
**Returns**: `WebConferenceParticipant` - Web conference participant object that is currently sharing (could also be myself)  
**Access**: public  

### webConferenceSession.getRemoteSharingParticipant() ⇒ `WebConferenceParticipant`

Get the remote web conference participant object that is currently sharing in the conference; IF the local participant is currently the sharing participant, will return null

**Kind**: instance method of [`WebConferenceSession`](#module_WebConferenceSession)  
**Returns**: `WebConferenceParticipant` - Web conference participant object that is currently sharing (not myself)  
**Access**: public

### webConferenceSession.isLock() ⇒ `boolean`

Check if the current web conference is locked. In case it's locked, no other user is allowed to join it

**Kind**: instance method of [`WebConferenceSession`](#module_WebConferenceSession)  
**Returns**: `boolean` - Return a boolean value indicating whether the web conference is currently locked  
**Access**: public  

### webConferenceSession.getParticipantsListWithDelegateCapability() ⇒ `Array`

Get the list of participants to whom we can delegate the current conference. Delegation means that we give the ownership of the conference to another user

**Kind**: instance method of [`WebConferenceSession`](#module_WebConferenceSession)  
**Returns**: `Array` - Return an array of WebConferenceParticipant to whom we can delegate the conference  
**Access**: public
