## Methods

### addUser()

> **addUser**(`user`): [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`any`\>

Add a user in the list. A ON\_USER\_ADDED\_TO\_LIST event is sent afterwards at the completion of the operation to notify about adding the user to the list.

#### Parameters

• **user**: [`User`](doc/sdk/webv2/lts/api/User/interfaces/User)

The user to add to users list

#### Returns

[`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`any`\>

---

### removeUser()

> **removeUser**(`user`): [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`any`\>

Remove a user from the list. A ON\_USER\_REMOVED\_FROM\_LIST event is sent afterwards at the completion of the operation to notify about removing a user from the list.

#### Parameters

• **user**: [`User`](doc/sdk/webv2/lts/api/User/interfaces/User)

Reference of the user to remove from the list

#### Returns

[`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`any`\>

---

### updateProfile()

> **updateProfile**(`profile`): [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`any`\>

Update the profile of the users list. Only the general information, i.e. the name, the description and the favorite status can be updated using this method. Users can only be added or removed from the list using methods 'addUser' and 'removeUser' respectively. A ON\_USERS\_LIST\_PROFILE\_UPDATED event is sent afterwards at the completion of the operation to notify about the list's profile update.

#### Parameters

• **profile**: [`UsersListProfile`](doc/sdk/webv2/lts/api/UsersLists/interfaces/UsersListProfile)

The profile information to be updated

#### Returns

[`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`any`\>
