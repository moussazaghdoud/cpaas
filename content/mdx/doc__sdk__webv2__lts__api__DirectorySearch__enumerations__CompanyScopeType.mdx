# Enumeration: CompanyScopeType

Specified how the company specified in the search option will be managed in the search scope.

## Enumeration Members

### COMPANY\_EXCLUDED

> **COMPANY\_EXCLUDED**: `"COMPANY_EXCLUDED"`

Refines the search to exclude users belonging to a specified company.

---

### LIMITED\_TO\_COMPANY

> **LIMITED\_TO\_COMPANY**: `"LIMITED_TO_COMPANY"`

Refines the search by specifying on which company the users are only searched.
