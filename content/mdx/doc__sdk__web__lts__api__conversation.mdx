# Documentation

Conversation model

LTS release 4.0.0Published on 2022-12-21

## ConversationSDK

Conversation model. Conversation is a representation of an interaction between a Rainbow user and another user, bubble or a call. A conversation never ends and all interactions done can be retrieved.

### ConversationSDK~id : `string`

The ID of the conversation.

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true  

### ConversationSDK~type : `number`

The type of the conversation. Value 0 refers to one-to-one conversation, 1 to a bubble conversation and 2 to a call.

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true  

### ConversationSDK~contactId : `string`

ID of the associated contact. In case of bubble conversation the value will default to 'null'.

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true  

### ConversationSDK~bubbleId : `string`

ID of the bubble that the conversation is linked to, if any present. In case of one-to-one conversation the value will default to 'null'.

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true  

### ConversationSDK~name : `string`

Name of the conversation - if conversation is of one-to-one type.

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true  

### ConversationSDK~name : `string`

Name of a bubble, if the conversation is attached to a bubble. Otherwise, defaults to null.

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true  

### ConversationSDK~creationDate : `Date`

Creation date of the conversation

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true  

### ConversationSDK~lastModificiationDate : `Date`

Last modification date (i.e. the date of the last message)

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true  

### ConversationSDK~isFavorite : `boolean`

Boolean indicating whether the conversation belongs to the user's favorites

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true  

### ConversationSDK~messages : `string`

An array representing the messages sent in the conversation

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true  

### ConversationSDK~lastMessage : `string`

The content of the last message sent in the conversation.

**Kind**: inner property of [`ConversationSDK`](#module_ConversationSDK)  
**Access**: public  
**Read only**: true
