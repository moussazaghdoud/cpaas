### Channel~visibility : `string`

Channel visibility. There are three possible options: "private" (only the owner may publish messages, add or remove users), "company" (company users may join or leave, may be found by searching by the users in the same company) or "public" (may be found by all the users, only allowed users can create a public channel).

**Kind**: inner property of [`Channel`](#module_Channel)  
**Access**: public  
**Read only**: true
