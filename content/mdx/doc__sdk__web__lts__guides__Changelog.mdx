# SDK for Web

Here is the list of the changes and features provided by the **Rainbow-Web-SDK**

## \[4.0.0-LTS\] - 2022-11-21

-   `contact model` - add new properties related to custom user profile management : `fileSharingEnabled`, `fileStorageEnabled`, `instantMessagesEnabled`, `bubblesEnabled`, `webrtcAudioEnabled`, `webrtcVideoEnabled`, `webrtcSharingEnabled`, `modifyProfileEnabled`;
-   `contact model` - Add also `professionalMobileNumber` and `professionalMobileNumberCan`
-   `pbxService` - Add new APIs: `subscribeToCCDEvents` and `getCurrentCCDAgentStatus` for the CCD Agent management

## \[3.4.0-STS\] - 2022-10-06

-   `conferenceRecording` - add conferenceRecording that allows to record web conferences on server side
-   `suggestionsService` - add suggestionsService that allows to find contact suggestions in the same company, i.e. "people you may know"
-   `BREAKING CHANGE` - Message model no longer contains `alternativeContent`, use `additionalContent` instead

## \[3.3.0-STS\] - 2022-08-25

-   `callsLogService` - the maximum number of call logs per request is 250 (both for `fetchCallLogs` and `fetchPaginatedCallLogs`)
-   `virtualBackground` - add virtualBackground Service to manage the virtual background for calls

## \[3.2.0-STS\] - 2022-06-14

-   General bug fixes and stability improvements

## \[3.1.3-STS\] - 2022-06-16

-   Fix HDMI Screens chosen as output device when plugged-in.

## \[3.1.2-STS\] - 2022-06-03

-   Fix dual channel headset issues; Fix STEREO microphone use cases

## \[3.1.1-STS\] - 2022-05-18

-   Fix problems with the audio devices management

## \[3.1.0-STS\] - 2022-05-16

-   `WebRTCService` - ### BREAKING CHANGE ### remove useSpeaker and useCamera and useMicrophone APIs (user should use the new deviceManagement service) ### BREAKING CHANGE ### remove setChromeExtensionIdForSharing API (and globally deactivate chrome extension for sharing, as no longer needed)
-   `deviceManagement` - ### BREAKING CHANGE ### Service to be used for the device management (audio input/output and video input devices)

## \[3.0.2-LTS\] - 2022-04-27

-   `CallsLogService` -- remove deprecated getAll() API

-   Fix problems with `signout` method that sometimes does not resolve the promise
