Allow to sign-on to Rainbow from a user account that already exists.

#### Parameters

• **login**: `string`

Rainbow hosted user login.

• **password**: `string`

Rainbow hosted user password.

• **rememberMe**: `boolean`

(default: false) When set to true, user token will be stored in local storage for further connections.

### setRenewedToken()

> **setRenewedToken**(`token`): [`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`void`\>

Use this method to update the OAuth token which is being used. It is obligatory to update the token after each renewing. If the token is not renewed, an event AUTH\_TOKEN\_EXPIRE will be emitted and the user will be disconnected

#### Parameters

• **token**: `string`

String representing the new AuthToken received after the renewing

#### Returns

[`Promise`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise)<`void`\>

A promise that resolve when token is renewed.
