**Returns**: `Promise.<any>` - An object of the result

**Category**: Rainbow RPCoverXMPP RPC Client

**Access**: public

**Nodered**: true

**Since**: 2.22.0

Param

Type

Description

headers

`Object`

The Http Headers used to web request.

rpcoverxmppserver\_jid

`string`

the jid of the http over xmpp server used to retrieve the HTTP web request. default value is the jid of the account running the SDK.

### rpCoverXMPPService.callRPCMethod(rpcoverxmppserver\_jid, methodName, params) ⇒ `Promise.<any>`

This API allows to send a request to call a rpc method to a bare jid. The result is the result of the method called on RPC server.  

**Kind**: instance method of [`RPCoverXMPPService`](#module_RPCoverXMPPService)

**Returns**: `Promise.<any>` - An object of the result

**Category**: Rainbow RPCoverXMPP RPC Client

**Access**: public

**Nodered**: true

**Since**: 2.22.0

Param

Type

Default

Description

rpcoverxmppserver\_jid

`string`

the jid of the rpc server used to retrieve the request. default value is the jid of the account running the SDK.  
**Note :** only a full JID (with resource) is possible to used.

methodName

`string`

`system.listMethods`

method name of the rpc shared method on server used to retrieve the request.  
default value is `system.listMethods` for listing the available methods on RPC server.  
use `system.methodHelp` with the name of a method as parameter to get informations about it.  
Use `system.methodSignature` with the name of a method as parameter to get it's signature.

params

`Array.<any>`

Array with the parameters for the RPC request. This Array will be splitted in each parameters to the method shared on RPC Server when it is ran.

### rpCoverXMPPService.addRPCMethod(methodName, methodCallback, methodDescription, methodHelp) ⇒ `Promise.<any>`

This API allows to expose an RPC method to requests from an XMPP party.  

**Kind**: instance method of [`RPCoverXMPPService`](#module_RPCoverXMPPService)

**Returns**: `Promise.<any>` - An object of the result

**Category**: Rainbow RPCoverXMPP RPC Server

**Access**: public

**Nodered**: true

**Since**: 2.22.0

Param

Type

Description

methodName

`string`

The name of the method to be added in RPC server. This is the name used by the RPC Client to call it.

methodCallback

`string`

The callback of the method to be added in RPC server.  
It is this method which is called when the SDK receives an RPC request for the linked methodName.  
The method signature is built from this parameter.

methodDescription

`string`

The description of the method to be added in RPC server

methodHelp

`string`

The help of the method to be added in RPC server
